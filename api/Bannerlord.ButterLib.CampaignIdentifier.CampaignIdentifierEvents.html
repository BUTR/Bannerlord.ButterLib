    Remarks

    This event is used to update information in the CampaignDescriptor assigned to the campaign. It should be fired when initial player character's gender, culture or birthplace are changed.

    Methods

    OnDescriptorRelatedDataChanged()

    Fires OnDescriptorRelatedDataChangedEvent.

    Declaration
    public void OnDescriptorRelatedDataChanged()
    Remarks

    Said event should be raised when there are changes to the information stored in CampaignDescriptor. Such information is: culture, gender, name, age, birthplace and names of parents of the initial player character, as well as player's clan name. Note that only gender, culture and birthplace are really determinative ones, others are just for the reference and changes to them could be safely ignored.
