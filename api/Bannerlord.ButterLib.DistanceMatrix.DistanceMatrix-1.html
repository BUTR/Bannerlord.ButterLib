    Syntax
    public abstract class DistanceMatrix<T> : DistanceMatrix where T : MBObjectBase
    Type Parameters
    T

    The type of objects for which the distance matrix should be calculated.
    Remarks

    DistanceMatrix<T> implements built-in calculation for the Hero, Settlement, Clan and Kingdom objects. For any other MBObjectBase subtypes custom EntityListGetter and DistanceCalculator methods should be provided using special constructor Create(Func<IEnumerable<T>>, Func<T, T, Single>) .
